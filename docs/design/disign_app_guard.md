# Design: Guardian App

Код: `frontend/apps/guardian-app/` (Vue 3 + Vite + TS). Ключевые экраны:
- Login/Register
- Dashboard (телеметрия, алерты, инсайты)
- Wards/WardDetail
- Devices
- Location map
- Settings

API-клиенты: `src/api/*.ts`, состояние: Pinia (`src/stores/*`), realtime: `src/services/realtime.ts`.
