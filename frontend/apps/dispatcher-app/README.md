# Dispatcher App

Веб-приложение для диспетчеров экстренных служб системы Care Monitoring.

## Технологии

- **Vue 3** с Composition API
- **TypeScript** для типобезопасности
- **Pinia** для state management
- **Vue Router** для навигации
- **Axios** для HTTP запросов
- **Leaflet** для карт
- **date-fns** для работы с датами
- **Vite** для сборки

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Приложение будет доступно на http://localhost:5174

## Сборка

```bash
npm run build
```

## Функциональность

### Страницы

1. **Dashboard** - Главная панель управления
   - Статистика вызовов
   - Критичные вызовы
   - Активные вызовы

2. **Calls** - Список всех вызовов
   - Фильтрация по статусу и приоритету
   - Пагинация
   - Быстрые действия

3. **Call Detail** - Детальная информация о вызове
   - Информация о подопечном
   - Медицинские данные
   - AI анализ
   - История вызова
   - Заметки

4. **Map** - Карта с геолокацией вызовов
   - Маркеры вызовов на карте
   - Информационные окна
   - Фильтрация по статусу

### Stores (Pinia)

- **auth** - Аутентификация и пользователь
- **calls** - Управление вызовами

### API

Все API запросы идут через централизованный клиент с автоматическим добавлением токенов и обработкой ошибок.

## Переменные окружения

Создайте файл `.env`:

```
VITE_API_URL=http://localhost:3000/api/v1
```

## Особенности

- Автоматическое обновление токенов
- Обработка ошибок
- Responsive дизайн
- Real-time обновления (каждые 5-10 секунд)
- Интеграция с картами
- Приоритизация вызовов
