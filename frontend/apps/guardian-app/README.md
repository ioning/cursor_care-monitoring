# Guardian App

Веб-приложение для опекунов системы Care Monitoring.

## Технологии

- **Vue 3** с Composition API
- **TypeScript** для типобезопасности
- **Pinia** для state management
- **Vue Router** для навигации
- **Axios** для HTTP запросов
- **Chart.js** для графиков
- **date-fns** для работы с датами
- **Vite** для сборки

## Структура

```
src/
├── api/              # API клиенты
├── components/       # Переиспользуемые компоненты
├── layouts/         # Layout компоненты
├── router/          # Роутинг
├── stores/          # Pinia stores
├── views/           # Страницы
├── App.vue          # Корневой компонент
└── main.ts          # Точка входа
```

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Приложение будет доступно на http://localhost:5174

## Сборка

```bash
npm run build
```

## Функциональность

### Страницы

1. **Dashboard** - Главная страница с обзором
   - Статистика (подопечные, устройства, алерты)
   - Последние алерты
   - Список подопечных

2. **Wards** - Управление подопечными
   - Список всех подопечных
   - Создание/редактирование подопечных
   - Детальная информация

3. **Ward Detail** - Детали подопечного
   - Текущие показатели здоровья
   - Графики телеметрии
   - Алерты подопечного
   - Привязанные устройства

4. **Devices** - Управление устройствами
   - Регистрация устройств
   - Привязка к подопечным
   - Статусы устройств

5. **Alerts** - Управление алертами
   - Фильтрация по подопечным, статусу, уровню
   - Детальная информация об алертах
   - Управление статусами

6. **Settings** - Настройки
   - Профиль пользователя
   - Настройки уведомлений

### Stores (Pinia)

- **auth** - Аутентификация и пользователь
- **wards** - Управление подопечными
- **devices** - Управление устройствами
- **alerts** - Управление алертами

### API

Все API запросы идут через централизованный клиент с автоматическим добавлением токенов и обработкой ошибок.

## Переменные окружения

Создайте файл `.env`:

```
VITE_API_URL=http://localhost:3000/api/v1
```

## Особенности

- Автоматическое обновление токенов
- Обработка ошибок
- Responsive дизайн
- Графики телеметрии
- Real-time обновления (планируется)
