## Технические детали

### Параметры изображений
- **Формат**: JPG/JPEG
- **Ширина**: 1920px (для фонов), 800px (для иллюстраций)
- **Качество**: q=80 (оптимальный баланс качества и размера)
- **Оптимизация**: Автоматическая оптимизация через Unsplash CDN

### Использование
- Все изображения загружаются через CDN Unsplash
- Изображения кэшируются браузером
- Поддержка lazy loading для улучшения производительности
- Responsive: изображения автоматически масштабируются

### Лицензия
- Все изображения из Unsplash имеют лицензию Unsplash License
- Разрешено использование в коммерческих и некоммерческих проектах
- Не требуется указание авторства (но приветствуется)
- Можно использовать, изменять и распространять

## Альтернативные источники

Если потребуется заменить изображения, можно использовать:

### Unsplash
- Поиск по тегам: `healthcare`, `medical`, `elderly care`, `monitoring`, `emergency`, `technology`
- URL формат: `https://images.unsplash.com/photo-{PHOTO_ID}?w={WIDTH}&q={QUALITY}`

### Pexels
- Поиск по тегам: `healthcare`, `medical`, `elderly`, `monitoring`, `emergency`
- URL формат: `https://images.pexels.com/photos/{PHOTO_ID}/pexels-photo-{PHOTO_ID}.jpeg?w={WIDTH}&q={QUALITY}`

## Рекомендации по оптимизации

1. **Кэширование**: Изображения кэшируются браузером, что ускоряет повторные загрузки
2. **Lazy Loading**: Для изображений ниже fold можно добавить `loading="lazy"`
3. **Responsive Images**: Использовать `srcset` для разных размеров экранов
4. **WebP формат**: При необходимости можно использовать WebP для лучшего сжатия

## Обновление изображений

Для обновления изображений:
1. Найдите подходящее изображение на Unsplash или Pexels
2. Скопируйте ID изображения из URL
3. Замените в коде на новый URL с нужными параметрами
4. Проверьте отображение на разных устройствах
