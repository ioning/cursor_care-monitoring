# Vision: Usage Scenarios

- Опекун (guardian): авторизация → просмотр дашборда (телеметрия, алерты, локация подопечных) → уведомления.
- Подопечный (ward): носимое устройство/моб. клиент отправляет телеметрию и SOS.
- Диспетчер: получает критичные алерты/вызовы, видит карту, статус карточки вызова.
- Администратор: управляет организациями (multi-tenancy), пользователями, тарифами, фичами.
- Биллинг: планы/подписки на уровне организации, ограничение функций при неоплате.
